Overview

The T1D Spatial Atlas is an interactive platform for exploring CosMX (NanoString) spatial transcriptomics data from human pancreatic tissues. It provides tools to visualize gene expression, explore spatial organization, and use AI to perform analysis across 20 donors representing four key stages of type 1 diabetes progression.

The platform includes a FOV Viewer for spatial exploration, a Gene Expression module for querying the 1,000-gene panel, an Image Viewer for tissue morphology, and an AI Chat interface for natural-language analysis.

Key Statistics
  • 1,139,248 cells profiled at single-cell resolution
  • 408 Fields of View (FOVs) across all donors
  • 1,000 genes measured per cell (CosMX panel)
  • Full spatial XY coordinates preserved for each cell
  • Cell type annotations from supervised classification
Data Access

The underlying data is available via controlled access through the NIDDK Central Repository. Researchers requiring raw data files should apply through the repository portal. The platform provides exploratory analysis without requiring download of raw files.

FOV Viewer

The FOV Viewer lets you navigate the spatial layout of individual fields of view. Use the control panel to:

  • Select a donor and specific FOV to display
  • Color cells by cell type, gene expression, or cluster
  • Adjust dot opacity for layered overlays
  • Zoom in/out to inspect islet structures
Gene Expression

Use the Gene Expression page to query any of the 1,000 panel genes. Enter a gene symbol (e.g. INS, CD8A, HLA-A) to see expression levels broken down by cell type and disease stage.

Expression values are normalized and log-transformed. Log2 fold change and adjusted p-values are computed via pseudobulk differential expression analysis.

AI Chat

The AI Chat interface allows natural-language queries about the dataset. Example questions:

  • Which genes are most upregulated in beta cells at T1D stage?
  • Compare islet composition between Control and AB⁺LN⁺
  • Show top markers for CD8⁺ T cells across all stages
  • What is the spatial distribution of STAT1 in insulitis?
FAQ
What sequencing platform was used?

Data was generated using the CosMX Spatial Molecular Imager (NanoString), enabling single-cell spatial profiling with a 1,000-gene panel.

How were cell types annotated?

Cell types were annotated using a combination of reference-based label transfer and manual curation of marker gene expression. Major cell types include beta, alpha, delta, PP, ductal, acinar, stellate, endothelial, and immune cells.

Can I download the raw data?

Raw data is available through controlled access via the repository portal. Contact us for access information.
